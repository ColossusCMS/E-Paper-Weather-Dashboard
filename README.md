# E-Paper 날씨 대시보드

## 1. 개요
### 1. 프로그램 개발환경
    - 개발 언어 : Python 3.12.2
    - 개발 툴 : VS Code
    - 개발 환경 OS : Windows 11
    - DB 구성 : MariaDB 10.6.22
  
### 2. 클라이언트 환경
    - OS : 라즈비안OS 커널 버전 6.12 (Debian 12)
    - 라즈베리파이 보드 버전 : 라즈베리파이 Zero 2W
    - 동작언어 : Python 3.11.2
    - 디스플레이 : 7.3 Inch E Ink Spectra 6(E6) Display (800x480)
